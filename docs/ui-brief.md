# 자비스 UI 컴셉 brief — Claude Design 입력용

> 사용자(대두)가 Claude Design 에 던질 컨텍스트.
> 두 층으로 분리: **고정(Must)** = 기능/정체성 / **열린(Open)** = UI·UX 판단 영역.
> 작성일: 2026-06-21 (Day 4)

---

## 한 줄

자비스 = 개발자의 개인 비서. Obsidian 위키를 long-term memory 로 쓰는 RAG 챗봇 + 음성 인터페이스. 데스크톱 앱(Electron). ⌘⇧Space 로 어디서든 호출.

---

## 고정 요구사항 (Must)

이 항목들은 빠지면 안 됨. UI/UX 방향에 관계없이 모두 반영.

### 기능
- 위키 RAG 기반 텍스트 채팅 (사용자가 질문하면 위키에서 근거 찾아 답변).
- 음성 입력: Whisper 로 STT. push-to-talk 또는 음성 모드 진입 시 자동 listen.
- 음성 응답: TTS 로 자비스가 말함 (음성 인터페이스에서만).
- 인용 페이지 `[[페이지명]]` 시각 강조 필수 — 클릭 가능한 칩/태그 형태로.
- 응답은 스트리밍 (토큰이 부드럽게 누적).
- Electron 데스크톱 앱. 메뉴바 상주(tray) + ⌘⇧Space 글로벌 단축키 토글.

### 상태 (음성 기능 필수)
음성 기능 동작 중 4가지 상태를 시각적으로 구분해야 함 (시각화 방식은 자유):
- `idle` — 대기
- `listening` — 사용자 말 듣는 중
- `thinking` — LLM 응답 생성 중
- `speaking` — TTS 재생 중

### 기술 제약
- Next.js 16 + Tailwind v4 + React 19. 시안 결과물은 이 스택과 호환되는 JSX + Tailwind 클래스로.
- 라이브러리는 가능하면 의존성 추가 없이. 필요 시 단순 SVG/Canvas 우선, Three.js 등은 cost 의식.

### 정체성 / 감성 (강제 아닌 영감)
- 영화 아이언맨의 자비스 같은 감성을 **음성 인터페이스 부분**에 입히고 싶음. HUD 풍, 다크 톤, 사이파이.
- 단순 텍스트 채팅 부분은 claude.ai 처럼 깨끗하고 가독성 좋은 톤.
- **이 두 감성은 영감일 뿐, 더 나은 UX 안 있으면 제안해줘. 자비스 풍이 시연 임팩트 부족하거나 텍스트/음성 톤 분리가 어색하다 판단되면 다른 방향도 OK.**

## 열린 결정 (Open) — Claude Design 의 판단 영역

아래는 사용자가 결정 안 했음. Claude Design 이 더 좋다 판단하는 방향으로 가도 됨. 시안 보면서 같이 조율.

- **텍스트 채팅 / 음성 인터페이스를 모드 분리할지, 한 화면에서 변형할지** (애니메이션 전환 등 더 좋은 안 있으면 제안).
- 레이아웃, 컴포넌트 구조.
- 컬러 팔레트 — 방향만 "다크 베이스 선호". 액센트 컬러는 자유 (시안/블루 자비스 풍 / Anthropic 호박색 / 그 외).
- 타이포그래피.
- 음성 인터페이스의 시각화 형태 — 회전 링 / 파동 / 구체 / 입자 / 그 외. 4가지 상태가 구분되기만 하면 자유.
- 음성 응답 텍스트 표시 방식 — 자막 / 채팅 / 큰 가운데 텍스트 / 그 외.
- 모드(또는 음성 인터페이스) 전환 트리거 — 버튼 / 단축키 / 마이크 길게 누르기 / 자동.
- 빈 상태, 로딩, 에러 상태의 시각 처리.

## Claude Design 에 던질 때 팁

1. 위 brief 통째로 컨텍스트로 던지고 "자비스 UI 컨셉이야. 위 고정 요구는 다 반영하되, 열린 결정은 너의 UX 판단 우선으로 가줘" 라고 명시.
2. 시안 요청은 단계별로 쪼개도 OK (디자인 토큰 → 컴포넌트 → HUD). 한 번에 다 받으려고 하면 디테일 떨어짐.
3. 시안 받을 때 검수 항목: 인용 페이지 칩 / 4가지 상태 시각화 / 음성 입력 진입로 / 단축키 안내 자리 — 빠지면 추가 요청.
4. 시안 결과물 가져올 때 가능하면 **컴포넌트 JSX + Tailwind 클래스** 형태로. 스크린샷만 가져오면 통합 시간 ↑.
5. 마음에 안 들면 "더 자비스스럽게" / "더 미니멀하게" / "이 부분만 다시" 같이 자유 피드백 — Claude Design 이 반복 다듬어줌.
